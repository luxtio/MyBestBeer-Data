# MyBestBeer Data Repository

This repository contains the database files for the MyBestBeer application.

## Contents

- db/beers.it.json - Italian beer database
- db/beers.en.json - English beer database  
- db/suggestions/ - Autocomplete suggestion files for various fields
- db/version.json - Version tracking for resources

## Usage

These files are served via jsDelivr CDN for lazy loading in the MyBestBeer app.

CDN Base URL: `https://cdn.jsdelivr.net/gh/luxtio/MyBestBeer-Data@v1.0.3/db/`
